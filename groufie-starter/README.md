## Quick start

## Deploy to Netlify
- Connect this repo in Netlify.
- Build command: `npm run build`
- Publish directory: `dist`
- Add custom domain: `app.groufie.co.za`

## Environment
Add your API base later, e.g.:
```js
export const API_BASE = 'https://api.groufie.co.za'
```
